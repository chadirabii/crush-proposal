## Run Locally

Clone the project

```bash
  git clone https://github.com/chadirabii/dev-finder
```

Go to the project directory

```bash
  cd Dev-Finder
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
